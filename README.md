# 🌊 Marine Client

A modern Minecraft launcher built to make playing modded Minecraft easier, faster, and more customizable.

> **⚠️ First Release**
>
> Welcome to the first public release of Marine Client! This is only the beginning. More features, improvements, performance optimizations, and quality-of-life updates will be added over time.

---

## ✨ Features

* 📦 Mod & Modpack Support
* 🔧 CurseForge & Modrinth Integration
* 👤 Built-in Account Switcher
* 🎨 Fully Customizable GUI
* 📜 Game Logs
* 🚀 Performance Booster
* 💾 Memory (RAM) Limiter
* 📥 One-Click Modpack Imports
* 🎭 Skin Changer
* 🕹️ Supports Minecraft **1.21.x and below**

---

## 📥 Installation

1. Download the latest release from the **Releases** page.
2. Run the installer.
3. Launch Marine Client.
4. Sign in with your Minecraft account.
5. Enjoy!
